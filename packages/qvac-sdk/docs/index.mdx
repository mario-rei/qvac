# QVAC SDK

**QVAC SDK** is the canonical entry point to develop AI applications with QVAC.

Written in TypeScript, it provides all QVAC capabilities through a unified interface, while also abstracting away the complexity of running your application in a JavaScript runtime other than Bare.

## Installation

[**Get started:**](/docs/sdk/get-started) installation requirements and procedures on all supported platforms, and usage example.

## Functionalities

### AI tasks

- [**Completion:**](/docs/sdk/completion) LLM inference via [`llama.cpp`](https://github.com/ggml-org/llama.cpp).
- [**Transcription:**](/docs/sdk/transcription) speech-to-text (ASR) via [`whisper.cpp`](https://github.com/ggml-org/whisper.cpp).
- [**Text embeddings:**](/docs/sdk/text-embeddings) via `llama.cpp`.
- [**Translation:**](/docs/sdk/translation) translate text between different languages.
- [**Text-to-Speech:**](/docs/sdk/text-to-speech) TTS via ONNX.
- [**Multimodal:**](/docs/sdk/multimodal) Multimodal via [`llama.cpp`] — i.e., process and understand multiple types of media within the same conversation context.
- [**RAG:**](/docs/sdk/rag) out-of-the-box retrieval-augmented generation workflow.
- [**Delegated inference:**](/docs/sdk/delegated-inference) perform peer-to-peer inference delegation via Holepunch stack, enabling resource sharing.

### Utilities

- [**Logging:**](/docs/sdk/logging) out-of-the-box logging from model to your client.
- [**Download Lifecycle:**](/docs/sdk/download-lifecycle) pause and resume model downloads.
- [**Blind relays:**](/docs/sdk/logging) establish peer connections through NAT/firewalls by routing traffic via relay nodes.
- [**Sharded models:**](/docs/sdk/sharded-models) download a model that is sharded into multiple parts.
