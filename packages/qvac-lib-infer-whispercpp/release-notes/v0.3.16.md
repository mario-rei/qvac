# QVAC Transcription Whisper Addon v0.3.16 Release Notes

This release updates the audio decoder dependency to use FFmpeg instead of GStreamer, providing a more streamlined and portable audio decoding solution.

## Features

The `@qvac/decoder-audio` dependency has been upgraded from v0.2.9 to v0.3.3. This major update switches the underlying audio decoding implementation from GStreamer to FFmpeg. FFmpeg provides better cross-platform support and eliminates the need for GStreamer system dependencies, simplifying deployment and reducing potential compatibility issues across different operating systems.

The `@qvac/util-transcription` dependency has been updated from v0.1.3 to v0.1.4. This update includes a complete cleanup of GStreamer-related code, replacing all `GSTDecoder` references with `FFmpegDecoder`. The transcription pipeline now exclusively uses FFmpeg for audio decoding, ensuring consistency with the updated decoder package.

## Other

These dependency updates are internal changes and do not affect the public API. Existing code using the transcription pipeline will continue to work without modifications.
