# QVAC Transcription Whisper Addon v0.3.15 Release Notes

This release improves Linux compatibility by switching to Ubuntu 22.04 for x64 builds, ensuring prebuilt binaries work on a wider range of Linux distributions. The build infrastructure has been streamlined with improved Vulkan SDK handling and optimized CI workflows.

## Features

The build system now uses Ubuntu 22.04 for Linux x64 builds instead of the previous GPU runner, ensuring compatibility with the current oldest LTS version's glibc. This change guarantees that prebuilt binaries will work on older Linux distributions that users may still be running, expanding the supported deployment environments. The integration test matrix has been expanded to include Ubuntu 22.04 alongside Ubuntu 24.04, providing comprehensive compatibility validation across different Linux environments.

Vulkan SDK installation has been improved in the prebuild workflow. The SDK is now properly downloaded from the official source and configured for both x64 and arm64 Linux architectures, with correct environment variables set for library paths and pkg-config. This ensures consistent GPU acceleration support across all Linux build targets and eliminates potential version mismatches.

## Other

Integration tests have been optimized by removing the unnecessary Vulkan SDK installation step. The SDK is only required during the build process, not for running tests, which reduces test setup time and complexity. The workflow now relies on mesa-vulkan-drivers for testing, which is sufficient for integration test execution.

The build workflow has been simplified by removing the custom vcpkg installation step that was previously needed for the GPU runner. With the switch to standard Ubuntu runners, the standard vcpkg configuration is now sufficient, reducing build complexity and maintenance overhead.
