# QVAC Transcription Whisper Addon v0.3.14 Release Notes

This release improves the CI/CD pipeline with better prebuild artifact handling, reduced binary sizes, and more reliable version publication tracking.

## Features

The prebuild workflow now strips debug symbols from native addon binaries on Linux and macOS platforms, resulting in significantly smaller prebuilt artifacts. This reduces download times and storage requirements for end users while maintaining full functionality.

## Other

The redundant Android artifact replication step was removed as it is no longer necessary.
