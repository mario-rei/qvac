# QVAC NMT Addon v0.3.5 Release Notes

This release focuses on build performance improvements, introducing compilation caching and smaller artifact sizes through debug symbol stripping.

## Performance

### Faster Compilation with ccache

Added ccache support for Linux ARM64 builds, significantly reducing compilation times for subsequent builds. The cache is persisted across CI runs and configured with 2GB max size and compression enabled.

### Smaller Artifact Sizes

Debug symbols are now stripped from release binaries, reducing the size of distributed artifacts without affecting runtime functionality.
