# QVAC NMT Addon v0.3.6 Release Notes

This release improves Linux binary compatibility by building on Ubuntu 22.04, enabling the addon to run on a wider range of Linux distributions including those with older glibc versions.

## Infrastructure

### Ubuntu 22.04 Build Support

Linux x64 builds now use Ubuntu 22.04 instead of Ubuntu 24.04 to ensure broader compatibility across Linux distributions. Users on Ubuntu 22.04 and other distributions with older glibc versions can now use the addon without encountering libc compatibility errors.

Key changes:
- Linux x64 builds moved from Ubuntu 24.04 to Ubuntu 22.04 for glibc compatibility
- Android builds remain on Ubuntu 24.04 as required for NDK tooling
- Integration tests now run on both Ubuntu 22.04 and 24.04 to verify cross-version compatibility
- Vulkan SDK configuration updated to use the appropriate sources for each Ubuntu version
