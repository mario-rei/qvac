# QVAC NMT Addon v0.3.7 Release Notes

This release adds a supported platforms section to the README, fixes the C++ unit test build, and includes several CI/infrastructure improvements.

## Documentation

### Supported Platforms

The README now includes a clear supported platforms table listing all Tier 1 platforms with their supported architectures and minimum OS versions: macOS (arm64/x64, 14.0+), iOS (arm64, 17.0+), Linux (arm64/x64, Ubuntu 22+), Android (arm64, 12+), and Windows (x64, 10+).

## Bug Fixes

### Unit Test Build Fix

Fixed a build issue with C++ unit tests that prevented them from compiling successfully.

### Debug Variable Cleanup

Removed unnecessary debug variables from the codebase, reducing noise in build output and improving code clarity.

## Infrastructure

### Linux ARM64 Build Compatibility

Linux ARM64 builds now use Ubuntu 22.04 to match x64 builds, ensuring consistent glibc compatibility across architectures.

### Release Notes Workflow Token

The release notes check workflow now has proper token configuration, ensuring it can run correctly in CI.
