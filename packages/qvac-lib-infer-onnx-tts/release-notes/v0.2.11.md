# QVAC TTS ONNX Addon v0.2.11 Release Notes

This release focuses on reducing binary size and improving compatibility when the addon is loaded alongside other native modules. Internal symbols are now hidden to prevent conflicts, and debug symbols are stripped from production builds.

## Features

### Reduced Binary Size

Debug symbols are now stripped from prebuilt binaries on Linux and macOS platforms. This significantly reduces the size of the distributed `.bare` files without affecting functionality.

### Symbol Visibility Improvements

Internal ONNX Runtime symbols are now hidden from the dynamic symbol table, exposing only the required `bare_*` and `napi_*` entry points. This prevents potential symbol conflicts when the addon is loaded in the same process as other native modules that may include their own copies of ONNX Runtime or other shared dependencies.

## Other

Removed the workflow step that was unnecessarily replicating android-arm64 artifacts to other Android architectures.
