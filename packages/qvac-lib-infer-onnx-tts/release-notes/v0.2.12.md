# QVAC TTS ONNX Addon v0.2.12 Release Notes

This release updates the build infrastructure to use Ubuntu 22.04 for Linux x64 builds, ensuring compatibility with the current oldest LTS version.

## Other

### Build System Updates

The Linux x64 build environment has been switched from Ubuntu 24.04 to Ubuntu 22.04. This change ensures that prebuilt binaries are compatible with systems running the current oldest LTS version of Ubuntu. The integration test suite now runs on both Ubuntu 22.04 and 24.04 to verify compatibility across these environments.
